Como usar este manual

Os guias são construídos para responder às dúvidas mais frequentes. Utilize a busca global no topo do sistema digitando a pergunta exatamente como ela surge no dia a dia. Quando houver um guia correspondente, a busca levará você diretamente para a página correta.

  • Links diretos para compartilhar com colaboradores ou parceiros.
  • Capturas de tela oficiais para garantir que todos sigam o mesmo fluxo.
  • Checklist de conferência ao final de cada procedimento.
